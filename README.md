#### Mini Makers

Este projeto é um sistema desenvolvido em **HTML, CSS e JavaScript**, com o objetivo de criar um lugar seguro, onde os pais possam entreter seus filhos com atividades fáceis e que estimulam a criatividade da criança e reforçam os laços entre a família.
No site, há a aba onde você encontra as atividades, formas de contato e um calendário interativo que voce pode fazer com seu filho.
